## KEYBOARD SHORTCUTS

	- Open Images browser: ALT + I

	- Open Regions: ALT + R

	- Open Notes: ALT + K

	- Open Export box: ALT + S

	- Open Load box: ALT + L

	- Crop Image (when crop button is triggered only): ALT + C

	- Show/Hide Menu: ALT + M
