Digital-Lightbox
================

Digital Lightbox is a virtual lightbox which allows the manipulation and the comparison of manuscript images.

##Requirements
  - Django 1.4
  - PIL

##Libraries used
	- JQuery and JQuery UI
	- Bootstrap
	- LESS
	- Resemble.js
	- Polyfill
	- JQuery Multidraggable
	- GTKNotifications
	- JCrop
	- CSS3 Mediaqueries
